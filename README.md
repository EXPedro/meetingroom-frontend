# ClientRoom

Este projeto foi desenvolvido com o [Angular CLI](https://github.com/angular/angular-cli), durante o *bootcamp* do banco Santander, ministrado pela Digital Innovation One, sob orientação da especialista professora [Kamila Santos](https://www.linkedin.com/in/kamila-santos-oliveira/).

Foi aprimorado o conhecimento no desenvolvimento de um projeto que realiza todas as principais operações CRUD.

Trata-se do *front-end* do sistema de agendamento de salas de reunião, cujo backend encontra-se [neste repositório](https://github.com/EXPedro/meetingroom).

## Development server

Em modo de desenvolvimento, tendo rodado o [backend](https://github.com/EXPedro/meetingroom), pode-se rodar o comando `ng serve` para iniciar um dev server, no endereço padrão do angular, `http://localhost:4200/`. Nele podem ser realizadas as operações de get (Room List), get by id (details), post (add Room), put (update) e delete.
